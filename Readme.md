## Gulpjs and Typrescript-React Project Bunde Example

## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` and `gulp-cli` installed globally on your machine.  

Installation:

`npm install`  

To Build App:  

`npm run build`  

To Start Server:

`npm run start`  

To Visit App:

`localhost:8000`
